# noderestserver
Simple Node REST Server
Welcome to the noderestserver wiki! 

## Installation 
1. Install node.js and npm 
2. Clone this Repository 
3. Go to the cloned Repository, and run the following: 
4. npm install
5. node app.js
6. Now your server should start
